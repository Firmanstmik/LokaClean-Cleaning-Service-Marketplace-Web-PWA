## 2. Facebook Pixel & Meta Setup
**Goal:** Retarget visitors on Instagram/Facebook.

### Steps:
1.  **Create Pixel:**
    *   Go to [Facebook Events Manager](https://business.facebook.com/events_manager).
    *   Click "Connect Data Sources" > "Web".
    *   Name it "LokaClean Pixel".
2.  **Install Code:**
    *   Copy the Pixel ID.
    *   Use `react-facebook-pixel` in the app (or manually add script to `index.html` head).
3.  **Verify Events:**
    *   Ensure `PageView` fires on all routes.
    *   Set up custom events for `CompleteRegistration` (on /register) and `Purchase` (on booking).
